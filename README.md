### What I build

**Developer tooling**
- [collie](https://github.com/suleymanozkeskin/collie) — Index-accelerated code search, an alternative to ripgrep/grep, optimized for both agents and humans. *(Rust)*
- [wcag-doctor](https://github.com/suleymanozkeskin/wcag-doctor) — WCAG 2.1 color-contrast compliance checker for React/Next.js projects. *(Rust)*
- [papagan](https://github.com/suleymanozkeskin/papagan) — Fast ISO 639-1 language detection, with npm and PyPI bindings. *(Rust)*

**Distributed systems**
- [horsies](https://github.com/suleymanozkeskin/horsies) — PostgreSQL-backed task queue and DAG workflow engine for Python.
- [horsies-rust](https://github.com/suleymanozkeskin/horsies-rust) — Rust port of horsies.

**Frontend**
- [shadstack-table](https://github.com/suleymanozkeskin/shadstack-table) — A shadcn/ui port of material-react-table. *(TypeScript)*
